# Signal Tool Starter

A clean starter repo for the three-engine interface:

- Meaning
- Origin
- Verification

## Files
- `index.html` — UI
- `style.css` — styles
- `script.js` — engine logic
- `reference-library.json` — assertion taxonomy and named record systems
- `SYSTEM_ARCHITECTURE.md` — architecture anchor

## Goal
Demonstrate one input running through three constrained outputs without turning the system into a chatty assistant.
